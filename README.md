# [ Projeto Pijaminhas]

> Este é o Backend de uma aplicação de um site de e-commerce de pijamas

## 🚀 Tecnologias
* **Frontend:** React / TypeScript
* **Backend:** Node.js / TypeScript
* **Database:** Prisma / PostgreSQL
